exo-3 (TOF):

>> pour le tester l'efficacite du code executer ces commandes: 

1. "open"   (creer un nouveau fichier et choisissez TOF pour le type)
2. "load"   (pour charger le fichier a un pourcentage pre)
3. "show"   (pour voir le contenu du fichier et reperer quelques elements)
4. "del"    (pour supprimer ces quelques elements)
5. "adjust" (pour effectuer l'ajustement du fichier et eliminer les elements logiquement supprime)
6. "show"   (pour voir le contenu du fichier apres ajustement)
7. "close"  (pour fermer le fichier en sauvegardant son entete)
